# Uptime — Student Digital Timetable App

## Product Identity
- **Name**: Uptime
- **Tagline**: Never miss a class. Stay ahead. Show up.
- **Category**: Student Productivity / Digital Timetable & Academic Time Management
- **Platform**: Web application (mobile-first, progressive design)

## Target Users
Tertiary institution students (university, college, polytechnic) who need to:
- Manage their weekly class schedules in one place
- Never miss or be late to a class
- Prepare effectively for tests and exams
- Start every study day feeling motivated

## Core Features

### 1. Class Timetable Setup
- Add classes with: subject name, day of the week, start time, room/location
- Weekly calendar view showing all scheduled classes
- Edit and delete classes easily
- Visual color-coded subjects for quick recognition

### 2. Smart Alarm Reminders
- Automatic notifications fire 1 hour AND 30 minutes before each class
- Two-stage advance warning system — enough time to prepare and travel
- Toggle reminders on/off per class
- Browser notification API integration

### 3. Study Timers
- Countdown timer for test and exam preparation sessions
- Customizable duration (e.g. 25 min, 45 min, 1 hour, custom)
- Visual countdown display with start/pause/reset controls
- Session label field (e.g. "Calc Exam Prep")

### 4. Word of the Day — Daily Motivational Pop-Up
- Appears every time the app is opened
- Curated motivational quote or encouragement message
- Rotating daily from a built-in collection of 30+ quotes
- Dismiss button to proceed to the dashboard

### 5. Dashboard
- Clean home view showing: today's classes, next upcoming class, quick-start study timer
- Motivational header greeting (e.g. "Good morning, let's crush today")

## Brand & Tone
- **Voice**: Empowering, modern, youthful, supportive — like a smart campus companion
- **Personality**: Practical, encouraging, always on time
- **Anti-references**: Avoid corporate/enterprise feel, cluttered UIs, dark/gloomy aesthetics
